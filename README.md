## Teste a aplicação

### STS
- git clone https://github.com/analudias/product-crud.git
- Import project into STS
- Run project

### Postman
- Create requests
  - POST new Produto
    - {{host}}/produtos/
  - PUT update Produto
    - {{host}}/produtos/1
  - DEL delete Produto
    - {{host}}/produtos/1
