# umCRUDporsemana

Fazendo um CRUD por semana até eu me sentir confiante que sei fazer CRUDs

## Teste a aplicação

### STS
- git clone https://github.com/analudias/product-crud.git
- Importe o projeto no STS
- Execute o project

### Postman
- Crie novas requests
  - POST new Produto
    - {{host}}/produtos/
  - PUT update Produto
    - {{host}}/produtos/1
  - DEL delete Produto
    - {{host}}/produtos/1
